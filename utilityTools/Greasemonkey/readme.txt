Greasemonkey関係のngスクリプト

==スクリプト説明

* remove_noused_Greasemonkey.ng
    使ってないGreasemonkeyスクリプトを削除・整理するNILScript | Web scratch
    http://efcl.info/2010/1021/res2008/

* GreasemonkeyToScriptish.ng
    How To: Manually transfer user scripts from Greasemonkey to Scriptish
     https://github.com/erikvold/scriptish/wiki/How-To%3A--Manually-transfer-user-scripts-from-Greasemonkey-to-Scriptish
    pref.jsに保存された値をGreasemonkeyからScriptishの書式へ変更する
    pref.jsと同じディレクトリにこのngスクリプトを置いて実行するとnew_pref.jsが生成される。
    (多分不完全なので手動でやった方が早いかもしれない。)
    convert pref.js form Greasemonkey to Scriptish.
    put the script into same directory of pref.js and excute.
