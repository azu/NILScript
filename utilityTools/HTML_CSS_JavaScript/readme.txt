JavaScript関係のツール

==スクリプト説明

* Clipboard_to_VimColorHTML.ng
    クリップボードのJavaScriptコードを http://vimcolor.spiritloose.net/ のサービスを通して
    シンタックスハイライトするためのHTMLコードに変換したものをクリップボードに格納。

* js_Beautifier.ng
    js_Beautifierを使ってクリップボードのJavaScriptを整形

* remove_htmlstyle.ng
    Office系からコピーした文章のHTMLはStyleがごちゃごちゃついて汚いので、
    Style属性以降を除去する。
    クリップボートにコピーした状態で使い、結果はクリップボードに上書きする
