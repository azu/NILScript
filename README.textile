[[NILScript|http://lukewarm.s151.xrea.com/nilscript.html]]上で動作するngファイルなどを置いています。

h2. 使い方

* [[NILScript|http://lukewarm.s151.xrea.com/nilscript.html]]をダウンロードする
* 使用したいngスクリプトをダウンロードする
* ng.exe(コンソールあり)とngw.exe(コンソールなし)のどちらかに、NILScriptのスクリプトファイルである*.ngファイルを渡す事で実行できる。
(通常ならngw.exeで実行するとよいと思います)
ngスクリプトをダブルクリックで実行できるように関連づけするにはinstall.batを実行する事で行えます。
